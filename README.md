# 404ButWeFound — Company Website

Modern, responsive, dark-theme landing page for **404ButWeFound** (web development company).

## Run locally

**Recommended (portfolio images will load):**

```bash
cd "c:\Users\shive\OneDrive\Desktop\COMPANY_WEBSITE"
python -m http.server 8000
```

Then open **http://localhost:8000/** in your browser.

If you double-click `index.html` instead, some browsers may block the portfolio images; use the server above if you don’t see them.

## Files

- `index.html` — page structure + content
- `styles.css` — theme + responsive layout + hover effects + animations
- `script.js` — sticky/mobile nav behavior, scroll-reveal, price calculator, contact form mailto
